## ğŸš€ Quick Start

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local server (optional, for development)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/eddy-bob/eddy-portfolio.git
   cd eddy-portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your browser, or
   - Use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (npx)
   npx serve
   
   # Using VS Code
   # Install "Live Server" extension and click "Go Live"
   ```

3. **View the site**
   - Navigate to `http://localhost:8000` (or your server's URL)
